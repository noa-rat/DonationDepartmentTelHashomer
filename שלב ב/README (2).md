## תיאור כללי
 **תרומות (Donations)** – כל תרומה נרשמת במערכת עם פרטים כמו סכום ומועד התרומה ואמצעי התשלום. תרומה יכולה להיות מיועדת לפרוייקט ספציפי או ניתן לפצל אותה בין המחלקות.
